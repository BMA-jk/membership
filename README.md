# Bhartiya Modi Army J&K Membership App

React + Vite + Tailwind + Supabase implementation with public registration, member portal with ID card download, and admin panel.

See `SUPABASE_SETUP.md` for database, RLS, storage and Edge Function configuration.

Images `card-bg.jpg` and `form-bg.jpg` should be copied into the `public/` folder (you can use your provided membership card and form designs).
